munkirjat-v3
============

Munkirjat.com using Scala and Play Framework

##Technology used
* Scala
* Play Framework
* Less
* JavaScript
* Bootstrap
* AngularJS

##Installation

1) `git@github.com:janimattiellonen/munkirjat-v3.git`

2) `munkirjat-v3`

3) `sbt`

4) `run`

5) browse to http://localhost:9000
